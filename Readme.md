## A Real-time Speech-to-Speech translation system

## Workflow
Audio (Person-A) -> Noise reduction -> VAD -> ASR -> MT -> text-to-audio -> audio (Person-B)

# Models
DeepFilterNet, Faster Whisper (Medium), NLLB

## API
gTT
   
## UI
frontend - HTML, CSS, Javascript / React Native

## backend 
FastAPI , WebSocket

## usage
It can be deployed in public spaces - hospitals, police stations, railway stations.
